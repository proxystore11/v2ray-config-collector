## ویژگی‌ها
- سبک و مینیمال، طراحی iOS  
- خروجی‌های TXT پاک و استاندارد  
- پشتیبانی از Hiddify / V2RayNG / Nekobox  
- بروزرسانی خودکار چندبار در روز  
- QR Code اختصاصی برای هر لینک  
- مناسب استفاده در کانال، وبسایت و ربات تلگرام
